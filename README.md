## âš™ï¸ Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/KavyaSakthivel/Tara-1.0.git
   cd Tara-1.0
   ```

2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   python app.py
   ```

4. Launch the frontend (optional, if separate):
   ```bash
   npm install
   npm start
   ```
